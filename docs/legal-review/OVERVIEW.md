# Kontaxt 서비스 개요 — 외부 검증 자료 v0.1

> 본 문서는 한국 가상자산 전문 세무사가 Kontaxt 의 양도소득 계산 엔진을 외부 검증할 때 가장 먼저 읽도록 작성한 1페이지 개요입니다. 패키지 전체 구성은 [README.md](README.md), 검증 시나리오는 [scenarios/](scenarios/) 폴더, 법령 정책은 [../tax-law-compliance.md](../tax-law-compliance.md) 를 참조해 주십시오.

---

## 1. 제품 한 줄

**Kontaxt 는 한국 거주자 개인의 가상자산 양도소득세 (2027-01-01 시행) 자동 정산 플랫폼입니다.** Upbit·Bithumb·Binance 등의 거래내역을 CSV·PDF·XLS 로 직접 업로드하면, 한국 세법 (소득세법 시행령 §88 — 거주자 총평균법) 에 따라 양도소득·산출세액·신고용 리포트를 자동 산출합니다.

## 2. 대상 사용자

| 구분 | 내용 |
|------|------|
| 주 사용자 | 한국 거주자 (개인) — 분리과세 기타소득 신고 대상 |
| 사용자 규모 | 다수 거래소·다수 코인 보유, 연간 양도차익 250만 원 이상 발생 가능자 |
| 비거주자 모드 | 시행령 §183⑥ 가상자산주소별 이동평균법 — 참고용 보존 |

## 3. 데이터 흐름

```
사용자 거래소 다운로드 (CSV·PDF·XLS)
        ↓
Kontaxt 업로드 (브라우저 직접, API 연동 없음)
        ↓
정규화 (UnifiedTransaction) — 통화·시간대·수수료 표준화
        ↓
세무 엔진 (lib/engine/) — 총평균법 + 의제취득가액 + 의제 50% + KST 기준
        ↓
결과 산출 (RealizedGain·TaxResult) + PDF·엑셀 리포트 출력
```

거래소 API 직접 연동은 사용하지 않습니다. 사용자가 거래소에서 직접 다운로드한 원본 파일을 업로드하는 방식으로, 거래소 제휴 의존도가 0 입니다.

## 4. 적용 세법

| 영역 | 적용 조항 | Kontaxt 처리 |
|------|----------|------------|
| 평가 방법 (거주자) | 시행령 §88① + §92②4호 (2025-02-28 개정) | **총평균법** 단일 — FIFO·이동평균 폐기 |
| 의제취득가액 | 시행령 §88② | 시행 전 보유분 = max(실매수가, 2026-12-31 시가) |
| 필요경비 의제 | 시행령 §88④⑤ | 의제 코인 매도가액 × **50%** — 평균단가·lot 추적·매수비용·매도수수료 모두 의제 50% 안에 포함 (보수 해석) |
| 환율 시점 | 시행령 §88③2호 | 일별 매매기준율 (분 단위 시세 불필요) |
| 기본공제·세율 | 시행령 §64의3② | 분리과세 20% + 연 250만 원 기본공제 |
| 지방세 | 지방세법 §93 | 소득세 × 10% (= 실효세율 22%) |
| 시행일 | 부칙 | 2027-01-01 00:00 KST — KST (UTC+9) 기준 판정 |

상세는 [../tax-law-compliance.md](../tax-law-compliance.md) v1.0 (2026-05-22 작성, 법령 원문 검증 완료) 참조.

## 6. 검증 의뢰 맥락

- **검증 대상**: 위 4번 표의 자동 처리 영역 — Kontaxt 의 계산 알고리즘이 명시한 법령 조항에 부합하는지
- **검증 범위 밖**: 특정 사용자의 실제 신고서 작성·제출, 사용자별 세무 자문
- **검증 시점**: 2026년 하반기 — 2027-01-01 시행 전 엔진 신뢰도 lock 단계
- **검증 결과 활용**: 검증 의견서를 받아 (a) 적정 평가 부분은 마케팅·온보딩에 인용, (b) 부적정·정정 권고는 엔진 업데이트 반영

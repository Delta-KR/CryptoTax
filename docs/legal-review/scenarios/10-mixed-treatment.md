# 시나리오 #10 — 부분 의제 + 실가 + 시행 전 보유 mix

| 항목 | 값 |
|------|---|
| **검증 포인트** | 한 사용자의 다중 코인 다중 처리 방식 동시 적용 (총평균법 + 의제경비 50% + 의제취득가액) |
| **적용 법령** | 소득세법 시행령 §88①②④⑤ · §64의3② |
| **난이도** | ⭐⭐⭐⭐ |
| **과세 연도** | 2027년 |
| **사용자 유형** | 거주자 (개인) |
| **거래소** | Upbit 단일 |
| **자산** | BTC (시행 전 보유 + 시행 후 매수, 일반) + ETH (의제 코인) + DOGE (의제 코인) |

> 이 시나리오는 한 사용자가 동시에 (1) 시행 전 보유 BTC 의 의제취득가액, (2) 시행 후 BTC 매수의 총평균법 합산, (3) ETH·DOGE 의 의제경비 50% 를 모두 적용받는 케이스를 검증합니다. 각 코인은 독립적으로 처리되어야 하며, 합계 양도소득에서 기본공제·세율을 일괄 적용합니다.
>
> **⚠ 의제 50% 의 매도 수수료 처리**: 시나리오 #2 의 검증 항목과 동일하게, Kontaxt 엔진은 의제 50% 가 매도 수수료를 포함한 필요경비 전부로 해석합니다. ETH·DOGE 매도 손익은 매도가액 × 50% 로만 산출하고 매도 수수료를 별도 차감하지 않습니다.

## 3. 산식 흐름

```
[BTC — 총평균법 + 의제취득가액]
  매수 #1 단가 = max(40M, 60M) = 60,000,000 (시행령 §88②)
  carry  = 0.05 × 60M + 4,000  = 3,004,000
  buy_27 = 0.05 × 70M + 3,500  = 3,503,500
  avg    = 6,507,500 / 0.10    = 65,075,000

  pnl(#5) = 9,000,000 − (0.10 × 65,075,000) − 9,000 = +2,483,500

[ETH — 의제 50% (§88④⑤) — 매도 수수료 포함 해석]
  pnl(#6) = 6,000,000 − (6,000,000 × 50%) = +3,000,000

[DOGE — 의제 50% (§88④⑤) — 매도 수수료 포함 해석]
  pnl(#7) = 150,000 − (150,000 × 50%) = +75,000

[합계]
  total_pnl    = 2,483,500 + 3,000,000 + 75,000 = 5,558,500
  taxable_base = max(0, 5,558,500 − 2,500,000) = 3,058,500
  income_tax   = 3,058,500 × 20% = 611,700
  local_tax    = round(611,700 × 10%) = 61,170
  total_tax    = 672,870
```

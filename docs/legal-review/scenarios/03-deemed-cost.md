## 3. 산식 흐름

```
[STEP 1] 의제취득가액 적용 (시행령 §88②)
  매수 #1: max(40,000,000, 60,000,000) = 60,000,000 KRW/BTC
  매수 #2: max(50,000,000, 60,000,000) = 60,000,000 KRW/BTC

[STEP 2] 시행 전 carry (2026-12-31 잔량)
  carry_cost   = (0.10 × 60M + 4,000) + (0.05 × 60M + 2,500) = 9,006,500
  carry_amount = 0.15 BTC

[STEP 3] 2027 총평균단가
  total_cost   = 9,006,500 + (0.05 × 70M + 3,500) = 12,510,000
  total_amount = 0.20 BTC
  avg_price    = 12,510,000 / 0.20 = 62,550,000

[STEP 4] 매도별 손익
  pnl(#4) = 8,000,000 − (0.10 × 62,550,000) − 8,000 = +1,737,000
  pnl(#5) = 4,500,000 − (0.05 × 62,550,000) − 4,500 = +1,368,000

[STEP 5] 세액
  total_pnl    = 3,105,000
  taxable_base = max(0, 3,105,000 − 2,500,000) = 605,000
  income_tax   = 121,000
  local_tax    = 12,100
  total_tax    = 133,100
```

## 4. 법령 매핑

| 단계 | 적용 조항 | 내용 |
|------|----------|------|
| STEP 1 (의제취득가액) | 시행령 §88② | "시행 전 보유분의 취득가액은 실제 취득가액과 2026-12-31 시가 중 큰 값" |
| STEP 2 (carry) | 시행령 §88① | 거주자별 총평균법 — 시행 전 잔량을 시행일 기초로 이월 |
| STEP 3 (평균단가) | 시행령 §88① + §92②4호 | carry + 신규 매수 합산 평균 |
| STEP 5 (세액) | 시행령 §64의3② + 지방세법 §93 | 분리과세 20% + 250만 공제 + 지방세 10% |

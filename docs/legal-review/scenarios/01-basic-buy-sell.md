## 2. Kontaxt 계산 결과 (기대값)

### 2-1. 총평균단가 산출

시행령 §88① + §92②4호 에 따라 거주자별로 총평균법을 적용합니다. 시행령 §88 의 매수 부대비용(수수료)은 취득가액에 포함합니다.

| 항목 | 값 |
|------|---:|
| 총 매수 거래금액 합계 | 11,500,000 KRW |
| 총 매수 수수료 합계 (취득가액 포함) | 11,500 KRW |
| 총 매수 부대비용 포함 가액 | **11,511,500 KRW** |
| 총 매수 수량 | 0.20 BTC |
| **총평균단가** | **57,557,500 KRW/BTC** |

**산식**: `(5,000,000 + 5,000) + (3,000,000 + 3,000) + (3,500,000 + 3,500) = 11,511,500 KRW`
**평균**: `11,511,500 ÷ 0.20 = 57,557,500 KRW/BTC`

### 2-2. 매도별 손익

매도 시점의 양도가액에서 (매도수량 × 총평균단가) + 매도 수수료를 차감합니다.

| 매도 # | 양도가액 (KRW) | 매도수량 (BTC) | 취득원가 (단가 × 수량) | 매도 수수료 | **양도손익** |
|--------|--------------:|--------------:|----------------------:|-----------:|------------:|
| 4 | 8,000,000 | 0.10 | 5,755,750 | 8,000 | **+2,236,250** |
| 5 | 4,500,000 | 0.05 | 2,877,875 | 4,500 | **+1,617,625** |

**산식 (매도 #4)**: `8,000,000 − (0.10 × 57,557,500) − 8,000 = 8,000,000 − 5,755,750 − 8,000 = 2,236,250`
**산식 (매도 #5)**: `4,500,000 − (0.05 × 57,557,500) − 4,500 = 4,500,000 − 2,877,875 − 4,500 = 1,617,625`

### 2-3. 연간 양도소득 합계

| 항목 | 값 (KRW) |
|------|---:|
| 매도 #4 손익 | +2,236,250 |
| 매도 #5 손익 | +1,617,625 |
| **2027년 양도소득 합계** | **+3,853,875** |

## 3. 산식 흐름 (한눈에)

```
[STEP 1] 총평균단가 산출
  total_buy_cost = Σ(매수 거래금액 + 매수 수수료)
                 = 11,500,000 + 11,500
                 = 11,511,500 KRW
  total_buy_amount = Σ(매수 수량) = 0.20 BTC
  avg_price = total_buy_cost / total_buy_amount
            = 11,511,500 / 0.20
            = 57,557,500 KRW/BTC

[STEP 2] 매도별 양도손익
  pnl(매도) = 양도가액 − (매도수량 × avg_price) − 매도 수수료
  pnl(#4) = 8,000,000 − 5,755,750 − 8,000 = +2,236,250
  pnl(#5) = 4,500,000 − 2,877,875 − 4,500 = +1,617,625

[STEP 3] 연간 합계
  total_pnl = Σ(매도별 손익) = +3,853,875

[STEP 4] 과세표준 (§64의3②)
  taxable_base = max(0, total_pnl − 2,500,000)
               = max(0, 1,353,875)
               = 1,353,875

[STEP 5] 산출세액
  income_tax = taxable_base × 20% = 270,775
  local_tax  = round(income_tax × 10%) = 27,078
  total_tax  = income_tax + local_tax = 297,853
```

| **적용 법령** | 소득세법 시행령 §88③2호 · §88① · §64의3② |

## 3. 산식 흐름

```
[STEP 1] 거래별 KRW 환산 (§88③2호)
  매수 #1: (4,000 + 4) USDT × 1,300 KRW/USDT = 5,205,200 KRW
  매수 #2: (2,500 + 2.5) USDT × 1,350 KRW/USDT = 3,378,375 KRW
  매도 #3:  7,000 USDT × 1,400 KRW/USDT = 9,800,000 KRW (수수료 9,800 별도)

[STEP 2] 총평균단가
  total_cost   = 5,205,200 + 3,378,375 = 8,583,575
  total_amount = 0.15 BTC
  avg_price    = 8,583,575 / 0.15 = 57,223,833.33 KRW/BTC

[STEP 3] 매도 손익
  pnl(#3) = 9,800,000 − (0.10 × 57,223,833.33) − 9,800 = +4,067,817

[STEP 4] 세액
  taxable_base = max(0, 4,067,817 − 2,500,000) = 1,567,817
  income_tax   = 313,563
  local_tax    = 31,356
  total_tax    = 344,919
```

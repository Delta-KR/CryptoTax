| **적용 법령** | 소득세법 시행령 §64의3② · §88① · §92²4호 |

> **⚠ 엔진 한계 명시**: Kontaxt 엔진은 **연도별 양도손익만 자동 산출** 합니다. 결손금 5년 이월 통산은 **사용자 또는 세무사가 수동 적용** 해야 합니다 (현재 미구현). 평균단가 carry (잔량 + 평균가) 는 자동 처리되지만, 음수 손익의 다음 해 차감은 별도 모듈이 필요합니다. 본 시나리오의 "결손금 carry" 표는 수동 적용을 가정한 결과입니다. 세무사 검증 시 (a) 산식이 정확한지, (b) 엔진에 carry 자동 처리를 추가하는 것이 적절한지 함께 의견 요청합니다.

## 3. 산식 흐름

```
[2027]
  avg     = 8,008,000 / 0.10 = 80,080,000
  pnl     = 5,000,000 − (0.10 × 80,080,000) − 5,000 = −3,013,000
  carry → 2028 = 3,013,000 (결손금)

[2028]
  avg     = 3,503,500 / 0.05 = 70,070,000
  pnl     = 5,000,000 − (0.05 × 70,070,000) − 5,000 = +1,491,500
  after_carry = 1,491,500 − 3,013,000 = −1,521,500
  carry → 2029 = 1,521,500

[2029]
  avg     = 3,003,000 / 0.05 = 60,060,000
  pnl     = 10,000,000 − (0.05 × 60,060,000) − 10,000 = +6,987,000
  after_carry  = 6,987,000 − 1,521,500 = +5,465,500
  taxable_base = max(0, 5,465,500 − 2,500,000) = 2,965,500
  income_tax   = 593,100
  local_tax    = 59,310
  total_tax    = 652,410
```

## 6. 부록

### 6-1. 첨부된 PDF 출력 범위 안내

본 시나리오의 `05-carry-over.pdf` 는 **2027년 한 해의 양도손익 산출 결과만** 표시합니다 (-3,013,000 손실, 세액 0원). Kontaxt 엔진이 연도별 산출만 지원하므로 PDF 도 단년 출력입니다. 본 문서 §2 의 3년 연도별 손익·결손금 carry 표는 사용자 또는 세무사가 **PDF 출력 + 수동 carry 적용** 으로 도출한 결과입니다. PDF 와 본 문서의 산출 범위 차이는 시나리오 §0 의 ⚠ 엔진 한계 명시와 부합합니다.

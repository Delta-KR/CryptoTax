# 시나리오 #2 — 의제 코인 매도 (필요경비 의제 50%)

| 항목 | 값 |
|------|---|
| **검증 포인트** | 의제 코인 매도 시 필요경비 50% 일괄 적용 + 평균단가·lot 추적 무시 |
| **적용 법령** | 소득세법 시행령 §88④⑤ · §88① (BTC 일반 거래) · §64의3② |
| **난이도** | ⭐⭐ |
| **과세 연도** | 2027년 |
| **사용자 유형** | 거주자 (개인) |
| **거래소** | Upbit 단일 |
| **자산** | BTC (일반) + ETH (의제 코인 지정) |

> 본 시나리오에서 ETH 는 시행령 §88④⑤ 의 의제 적용 대상 코인으로 지정했다고 가정합니다. 의제 코인 매도 시 필요경비는 매도가액의 50% 로 일괄 적용하며, 평균단가·lot 추적은 무시합니다.
>
> **⚠ 핵심 해석 (검증 요청)**: Kontaxt 엔진은 의제 50% 가 **매도 수수료를 포함한 필요경비 전부** 로 해석합니다. 즉, 의제 코인 매도 손익 = 매도가액 − 매도가액 × 50%. 매도 수수료는 별도 차감하지 않습니다 (이중 차감 방지, 보수적 해석). 이 해석이 시행령 §88④⑤ 의 입법 취지에 부합하는지 의견 요청합니다.

## 3. 산식 흐름

```
[BTC — 총평균법]
  avg_BTC = (5,005,000 + 3,003,000) / 0.15 = 53,386,666.67 KRW/BTC
  pnl(#5) = 8,000,000 − (0.10 × 53,386,666.67) − 8,000 = +2,653,333

[ETH — 의제 50% (§88④⑤) — 매도 수수료 포함 해석]
  pnl(#6) = 12,000,000 − (12,000,000 × 50%) = +6,000,000
  pnl(#7) =  7,000,000 − ( 7,000,000 × 50%) = +3,500,000

[합계]
  total_pnl    = 2,653,333 + 6,000,000 + 3,500,000 = 12,153,333
  taxable_base = max(0, 12,153,333 − 2,500,000)   =  9,653,333
  income_tax   = round(9,653,333 × 20%)           =  1,930,667
  local_tax    = round(1,930,667 × 10%)           =    193,067
  total_tax    =                                     2,123,734
```

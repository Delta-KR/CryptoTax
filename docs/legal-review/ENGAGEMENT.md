## 제2조 (검증 업무 개요)

을은 갑이 운영하는 가상자산 양도소득세 계산 서비스 (이하 "Kontaxt") 의 계산 엔진 알고리즘이 한국 소득세법 시행령 §88·§92·§64의3 및 지방세법 §93 등 관계 법령에 부합하는지에 대한 외부 검증 업무를 수행합니다.

| 보안·개인정보보호 (PIPA) 검증 | 별도 검증 영역 |

## 제4조 (검증 산출물)

을은 다음 산출물을 갑에게 제출합니다.

1. 시나리오 10건 각각의 §5 검증 의견서 (단계별 정합성 + 법령 적용 적정성 + 종합 의견 + 등급)
2. 종합 의견서 ([OPINION_FORM.md](OPINION_FORM.md) 양식 기입본)
3. 발견 사항 정정 권고 (BLOCKING / HIGH / MEDIUM / LOW / NIT 분류)
4. 종합 등급 (PASS / CONDITIONAL PASS / FAIL)

산출물의 형식은 PDF 또는 서명·날인된 종이 문서 중 갑·을 협의로 정합니다.

## 제5조 (검증 일정)

| 단계 | 일정 | 주체 |
|------|------|------|
| 자료 송부 (본 합의서 + NDA + 패키지 일체) | D | 갑 |
| 1차 회신 (검증 가능 여부·일정 확인) | D+3 | 을 |
| 검증 수행 (시나리오 10건 + 종합 의견) | D+3 ~ D+14 | 을 |
| 산출물 1차 초안 제출 | D+14 | 을 |
| 갑 1차 검토 + 추가 질의 | D+14 ~ D+17 | 갑 |
| 산출물 최종본 제출 | D+21 | 을 |
| 보수 정산 | D+21 ~ D+28 | 갑 |

총 소요: **약 3~4주**. 갑·을 협의로 단축 또는 연장 가능.

## 제6조 (검증 보수)

| 항목 | 금액 (KRW, 부가세 별도) |
|------|------------------------:|
| 기본 검증 보수 (시나리오 10건 + 종합 의견서) | **400,000원** (가이드 — 갑·을 협의로 조정) |
| 추가 시나리오 (1건당) | 30,000원 |
| 갑의 추가 질의 응대 (시간당) | 50,000원 |
| 검증 완료 후 엔진 업데이트 재검증 (희망 시) | 별도 합의 |

> 본 가이드 금액은 한국 세무사 평균 자문료 (시간당 5~10만원) 와 검증 예상 소요 (3~5시간) 를 기준으로 산정한 참고치입니다. 을의 경력·전문성·실 소요 시간에 따라 조정 가능합니다.

### 6-1. 결제 조건

| 항목 | 내용 |
|------|------|
| 결제 시점 | 산출물 최종본 제출 후 7일 이내 |
| 결제 방법 | 계좌이체 |
| 세금계산서 발행 | 을이 부가세 별도로 발행 |

## 제7조 (책임 한계)

1. 을의 검증 의견은 본 합의서 체결 시점의 관계 법령 및 갑이 제공한 자료를 기준으로 작성됩니다.
2. 검증 의견은 갑의 엔진 알고리즘 개선·홍보 등의 내부·외부 활용을 위한 참고 자료이며, 특정 사용자의 신고서에 대한 책임을 지지 않습니다.
3. 본 합의서 체결 후 관계 법령이 개정되거나 갑이 엔진을 수정한 경우, 본 검증 의견은 그 범위에서 효력을 상실하며 별도 재검증이 필요합니다.

## 제8조 (의견서 활용)

검증 의견서의 활용 범위는 [OPINION_FORM.md](OPINION_FORM.md) 부록 의 "의견서 활용 합의" 에 따릅니다.

## 제9조 (해지)

1. 갑·을 일방의 귀책 사유로 본 합의서 이행이 불가능한 경우, 상대방은 본 합의서를 해지할 수 있습니다.
2. 해지 시 이미 수행된 부분에 대해서는 갑·을 협의로 정산합니다.

## 제10조 (관할 및 준거법)

본 합의서와 관련된 분쟁은 대한민국 법령을 준거법으로 하며, 갑의 주소지를 관할하는 법원을 1심 관할로 합니다.

## 제11조 (효력 발생 및 서명)

본 합의서는 양 당사자가 서명·날인한 날부터 효력이 발생하며, 본 합의서의 사본 1부씩을 갑과 을이 각각 보관합니다.
